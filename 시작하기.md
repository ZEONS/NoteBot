# EASYSAFE NoteBot - 시작하기

이 프로젝트는 Google Gemini API를 활용하여 사용자가 업로드한 노트를 기반으로 답변하는 지능형 지식 비서 챗봇입니다.

## 🌟 주요 기능
- **노트 기반 Q&A**: `my_notes` 폴더에 있는 PDF, Markdown, Text 파일을 분석하여 답변합니다.
- **다양한 파일 지원**: PDF 문서 내 텍스트 추출 기능을 포함합니다.
- **실시간 채팅 인터페이스**: 현대적이고 직관적인 웹 UI를 제공합니다.
- **설정 관리**: 웹 인터페이스에서 직접 API 키와 모델을 변경할 수 있습니다.

## 📂 프로젝트 구조
- `server.py`: FastAPI 기반의 메인 백엔드 서버입니다. (포트: 8001)
- `static/`: 웹 프론트엔드 파일들이 포함되어 있습니다 (`index.html`, `app.js`, `style.css`).
- `my_notes/`: 챗봇이 참조할 지식 소스(PDF, MD, TXT)를 저장하는 폴더입니다.
- `app.py`: Streamlit 버전의 대체 인터페이스입니다 (옵션).
- `.env`: API 키와 기본 모델 설정을 저장하는 파일입니다.
- `requirements.txt`: 프로젝트 실행에 필요한 Python 라이브러리 목록입니다.
- `지식비서_실행.bat`: 로컬 환경에서 서버를 즉시 실행하기 위한 배치 파일입니다.

## 🚀 실행 방법

### 1. 로컬 환경
- **간편 실행**: `지식비서_실행.bat` 파일을 더블 클릭합니다.
- **수동 실행**:
  ```bash
  pip install -r requirements.txt
  python server.py
  ```
- 브라우저에서 `http://localhost:8001` 주소로 접속합니다.

### 2. 서버 환경 (AWS EC2)
- 현재 서버(`13.124.208.217`)의 `/home/ec2-user/notebot` 경로에 배포되어 있습니다.
- 백그라운드에서 실행 중이며, `http://13.124.208.217:8001`에서 접속 가능합니다.

## ⚙️ 설정
- `.env` 파일에 `GEMINI_API_KEY`를 설정해야 합니다.
- 웹 UI 내의 "설정 및 모델 관리" 메뉴에서도 직접 수정이 가능합니다.
